## 🌟 核心功能地图

### 1. 📊 审核效率实时统计 HUD
*   **状态常驻**：页面右下角悬浮 HUD 球（支持双击在“迷你圆球”与“展开详情条”之间自由切换，且位置支持拖拽记忆）。
*   **多维度看板**：实时计算今日初审数、总核单量、剩余达标差值、以及基于订单时间间隔计算的**真实无稀释时速**。
*   **打卡达标特效**：今日初审审核量达到预设目标（如 250 单）时，屏幕自动喷洒彩带洒花特效 (`canvas-confetti`)，增添过单仪式感。
*   **数据大屏**：按下 `Alt + S` 弹出玻璃拟态暗黑风的统计面板，支持查询历史每小时统计柱状图、近 7 日趋势折线图，并提供一键导出 CSV 数据报表。

### 2. ⚡ 一键通过审核 (Alt + A)
*   **全自动过单**：按下 `Alt + A` 或点击悬浮“一键通过审核”按钮，脚本将自动识别各题目，点击默认的合规选项，并完成最终提交，秒速进入下一单。
*   **智能防御**：遇到您手动选择“不通过”的题目，一键通过逻辑会自动跳过该题，绝不覆盖您的手动审核选择。

### 3. 🖼️ Q5 ➔ Q6 照片证据同步克隆
*   **无缝联动**：Q6 题目中会自动将 Q5 的照片证据以 100% 原始尺寸克隆在右侧的“审核参考”栏中。
*   **右键复制支持**：复制过来的图片完美继承了平台原生的右键菜单，您可以在右侧直接“右键复制图片”，并在左侧“右键粘贴”上传，免去了滚屏回 Q5 复制的繁琐。

### 4. ❌ Q6 一键不通过 (快捷输入)
*   **智能红框检测**：点击 Q6 旁的“一键不通过”按钮，脚本会通过图片 URL 特征（是否包含 `annotation` 字符）智能检测出您已经画红框标注了第几张图片。
*   **自动理由生成**：自动生成类似 `第一张，第二张照片标注处补拍规格` 的文字，并填入原因对话框。
*   **极速确认**：在不破坏和影响您手动选择的单选原因的前提下，自动点击“确认”提交，审核速度提升 500%。

### 5. ⏱️ 间隔积分时速算法
*   **零稀释手速计算**：弃用了传统“按钟点均匀平摊”导致的时速稀释。通过分析您的连续审核时间间隔，自动扣除摸鱼和开工前的闲置时间，真实还原您的瞬时审核时速（以 5 分钟间隔为断点，不足 2 分钟时段自动平滑防抖）。
